ScrapeOpen is a collaborative project. It curates the publication of code to scrape websites serving public data sets and presents them in a format accessible to humans and machines.

What's public data set

A public data set is a collection of observations collected by a public agency (not-for-profit) and having a public value. In order to be published, the data set must not be proprietary.

What is an accessible format

Data sets published by ScrapeOpen project have at least 3 stars in Tim Berners-Lee’s 5-star Open Data plan

  • ★
    • make your stuff available on the Web (whatever format) under an open license
  • ★★
    • make it available as structured data (e.g., Excel instead of image scan of a table)
  • ★★★
    • make it available in a non-proprietary open format (e.g., CSV instead of Excel)
  • ★★★★
    • use URIs to denote things, so that people can point at your stuff
  • ★★★★★
    • link your data to other data to provide context

How you can help

  • Indicate interesting resources
  • Write the code to scrape a resource
  • Control the quality of the data
